# 单发对齐更新（调试流程 + 校准流程）

更新时间：2026-03-31

## 调试流程对齐

- 统一执行顺序：`E6 -> 驱动检查 -> A3 -> AC -> A3 -> BA`。
- `F4` 职责统一为“APD工作点确定”，不混入测距判断。
- 单发（电混）不做反光塞调节；双发/双收可调，但输出口径一致。
- `FB=0` 已定位：`S3 zero_boundary` 命中触发 `S4 blind_zero` 清零。

## 校准流程对齐

- 统一流程：`F4 -> F1(5000mm) -> FB`。
- `F1` 5000mm 参数：`00 00 13 88`（大端）。
- `F1` 仅 `verify pass` 才持久化；`verify fail` 不写入新校准。
- `F4` 成功后会写工作系数与 `dual_vbr/work_apd` 相关数据。
- 运行测距主要使用运行态 `APD/VBR` 比值与 VBR/工作点，不是只看 `coeff.apdwork` 单字段。
