# 三原理差异速览

| 维度 | 单发 (Single) | 双发 (Dual-TX) | 双收 (Dual-RX) |
|---|---|---|---|
| 原理 | 单发射/单接收口径 | 双发射分时切换 | 双接收并行采样 |
| 采样方式 | 双ADC电混合语义 | 单ADC + `LD_SELECT` 分时 | 双ADC同步采样 |
| `USE_DUAL_ADC_SYNC` | 1 | 0 | 1 |
| `USE_LD_SELECT` | 0 | 1 | 0 |
| 光路获取 | 内外语义由电混合/算法映射 | 通过 `LD_SELECT` 切内外 | 内外两路物理独立 |
| 关键调试风险 | 电混合口径误判 | 切换时序、路径映射错误 | 双路归一化/融合偏差 |

备注：
- 双发、双收已接入 unified 并可编译切换；
- 当前主线为单发口径在 unified 中的闭环调试。

## 统一执行原则（2026-03-31 同步）

1. 抽象原则
- 统一入口：业务层只通过 `hw_system_request(START/CONFIG/STOP)` 驱动硬件会话。
- 分层边界：驱动层负责单驱动状态机，会话层负责跨驱动顺序与 ref_count。

2. 调试原则
- 固定顺序：`E6 -> 驱动检查 -> A3(默认APD) -> AC -> A3(最佳APD) -> BA`。
- E6 通过标准：协议帧 + 运行日志双证据。

3. 校准原则
- 固定顺序：`F4 -> F1(5000mm) -> FB`。
- 职责边界：`F4` 工作点、`F1` 相位校准、`FB` 运行验证。
